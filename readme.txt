== Description ==
WP-Twitter is a plugin that creates a integration between your WordPress blog and your <a href="http://twitter.com">Twitter</a> account by giving you the following functionality: 
<ul>
<li> Post a tweet from the WordPress Admin Screens, including option to reduce the URL with API <a href="http://is.gd">is.gd</a> before sending. </li>
<li> Widget for displays yours latest tweets in your WordPress blog.</li>
</ul>

== Installation ==
<ol>
   <li>Download the plugin archive and expand it</li>
   <li>Activate the 'WP-Twitter' plugin in your WordPress admin 'Plugins'</li>
   <li>Go to the 'WP-Twitter Options page' (Options > WP-Twitter ») to 	set your Twitter account information.</li>
   <li>Go to 'Presentation / Widgets' in your WordPress admin area.</li>
   <li>Drag the 'WP-Twitter' widget to your sidebar.</li>
</ol>
<ul>
 Configure the options:
      <li> Account: your Twitter account name *required</li>
      <li>Title: the heading you want to appear above your Twitters in 	the sidebar</li>
      <li>Tweets: the number of Twitters shown.</li>
</ul>
